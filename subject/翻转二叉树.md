## 题目

翻转一棵二叉树。

示例：

输入：
```
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```

输出：
```
     4
   /   \
  7     2
 / \   / \
9   6 3   1 
```
## 解法

### 具体实现

| ------ | ----------------------------------------------------------------------------------------------- |
| 递归   | [invertTree](https://github.com/whiteCcinn/leetcode-practice/blob/master/algorithm/inverTree.c) |

### 解法一：递归

利用分治的思想，把问题划为更小的问题，从而利用递归实现。
